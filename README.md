# Agentic AI Assistant

A Flask-based Agentic AI Assistant powered by Ollama.

## Features

- Weather information
- Stock price lookup
- System command execution
- Local LLM using Ollama

## Technologies Used

- Python
- Flask
- Ollama
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Open:

http://127.0.0.1:5000
